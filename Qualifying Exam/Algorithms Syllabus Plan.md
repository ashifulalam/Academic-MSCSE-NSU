# Algorithms Syllabus Plan

This checklist breaks the Algorithms qualifying exam syllabus into smaller parts.

Use it to finish one small topic at a time.

## Part 1: Algorithm Basics

- [x] Understand what an algorithm is
- [ ] Understand flowcharts
- [ ] Learn best case running time
- [ ] Learn worst case running time
- [ ] Practice running time analysis using Insertion Sort

## Part 2: Asymptotic Analysis

- [ ] Learn Big O notation
- [ ] Learn small o notation
- [ ] Learn Big Omega notation
- [ ] Learn small omega notation
- [ ] Learn Big Theta notation
- [ ] Compare running time of different algorithms
- [ ] Practice worst case running time analysis

## Part 3: Divide and Conquer

- [ ] Understand divide and conquer strategy
- [ ] Study Merge Sort
- [ ] Analyze Merge Sort running time
- [ ] Study Quick Sort
- [ ] Analyze Quick Sort running time
- [ ] Solve recurrence relations using substitution method
- [ ] Solve recurrence relations using recursion tree method

## Part 4: Heap and Heap Sort

- [ ] Understand heap data structure
- [ ] Learn MaxHeapify
- [ ] Learn BuildMaxHeap
- [ ] Study HeapSort
- [ ] Analyze running time of heap operations
- [ ] Analyze running time of HeapSort

## Part 5: Sorting Algorithms

- [ ] Understand lower bound for comparison sorting
- [ ] Study Counting Sort
- [ ] Analyze Counting Sort running time
- [ ] Study Radix Sort
- [ ] Analyze Radix Sort running time
- [ ] Understand when linear time sorting is applicable

## Part 6: Dynamic Programming

- [ ] Understand dynamic programming
- [ ] Learn important properties of dynamic programming
- [ ] Compare dynamic programming with greedy approach
- [ ] Study Longest Common Subsequence
- [ ] Analyze LCS algorithm
- [ ] Practice Longest Common Subsequence problems

## Part 7: Graph Algorithms

- [ ] Study BFS
- [ ] Study DFS
- [ ] Study Topological Sort
- [ ] Study Kruskal's MST algorithm
- [ ] Study Prim's MST algorithm
- [ ] Study Dijkstra's algorithm
- [ ] Study Bellman-Ford algorithm
- [ ] Understand implementation details
- [ ] Understand application areas
- [ ] Analyze running time of graph algorithms

## Part 8: Complexity Theory

- [ ] Understand solvable and unsolvable problems
- [ ] Understand tractable and intractable problems
- [ ] Understand P vs NP
- [ ] Learn basic theory of NP-completeness
- [ ] Practice proving NP-completeness
- [ ] Understand approximation algorithms
- [ ] Understand approximation ratio

## Reference Books

- Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, `Introduction to Algorithms`
- Ellis Horowitz and Sartaj Sahni, `Computer Algorithms`
- Christos Papadimitriou, Umesh Vazirani, and Sanjoy Dasgupta, `Algorithms`
- Eva Tardos and Jon Kleinberg, `Algorithm Design`
